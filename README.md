# 🔐 亚马逊门锁售后分析系统

亚马逊门锁售后工单数据分析仪表盘，支持 Excel 上传、多维度分析和可视化展示。

## 功能

- **数据上传** — 支持 Excel 售后工单和销量数据上传
- **概况仪表盘** — 总订单量、售后工单量、售后率、涉及SKU种数
- **趋势图表** — 工单趋势、售后数量趋势、售后原因 TOP10、责任方分布
- **售后分析** — 标签分布、售后数量 TOP30 SKU
- **明细数据** — 可搜索/排序/筛选的工单明细表
- **年度筛选** — 按年份独立查看数据

## 技术栈

- Chart.js 4.4.0 — 图表渲染
- Dexie.js 4.0.4 — IndexedDB 本地存储
- SheetJS (xlsx) 0.18.5 — Excel 解析

## 本地运行

```bash
python -m http.server 8000
# 访问 http://localhost:8000
```

> 注意：需要 HTTP 服务器运行，不能直接双击 index.html 打开（file:// 协议限制）。
